backrub
    by Jan Krutisch
    http://halfbyte.github.com/backrub

== DESCRIPTION:

backrub is a very simple tool to set up a backup strategy for a server that
only has access to an FTP backup store. This is a common setup with cheap root server plans, such as the german Hetzner servers.

== FEATURES:

* simple strategies for daily backups (more complex strategies planned) of dynamic content, such as git, svn, trac repos, databases etc.
